# 8.5 Optional: Using the Contact Form 7 Plugin in WordPress

These steps explain how to install the plugin, add a contact form to a page, and customize the form with additional fields.

---

## 1. Install the Contact Form 7 Plugin

1. Log in to your **WordPress dashboard**.
2. In the left menu, go to **Plugins → Add New**.
3. In the search bar, type **Contact Form 7**.
4. Find the plugin named **Contact Form 7**.
5. Click **Install Now**.
6. After installation, click **Activate**.

Once activated, a new **Contact** menu appears in the WordPress dashboard.

---

## 2. Use the Default Contact Form

Contact Form 7 automatically creates a basic contact form.

1. Go to **Contact → Contact Forms**.
2. You will see a form called **Contact form 1**.
3. Copy the **shortcode** shown beside the form.

Example:

```
[contact-form-7 id="123" title="Contact form 1"]
```

---

## 3. Add the Contact Form to a Page

1. Go to **Pages → Add New**.
2. Create a page called **Contact Us** (or any title you prefer).
3. Add a **Contact Form 7** block to the page and choose the form.
4. Click **Publish**.

When you view the page, the contact form will appear.

The default form includes these fields:

* Name
* Email
* Subject
* Message

---

## 4. Editing the Contact Form

To modify the form:

1. Go to **Contact → Contact Forms**.
2. Click **Edit** on the form you want to change.

You will see several tabs.

---

## 5. Form Tab (Form Fields)

The **Form** tab controls the fields visitors see on the page.

You can add or edit fields such as:

* Text fields
* Email fields
* Dropdown menus
* Checkboxes
* Radio buttons
* File uploads

Each field is created using a **tag**, which looks similar to a shortcode.

---

## 6. Mail Tab (Email Settings)

The **Mail** tab controls how the email is sent when someone submits the form.

Important settings include:

### To

The email address that receives the form submission.

### From

The email sender that appears in the message header.

### Reply-To

Usually set to the visitor’s email address so you can reply directly.

### Subject

The subject line of the email.

### Message Body

The content of the email that includes the form fields.

Example body:

```
Name: [your-name]
Email: [your-email]
Subject: [your-subject]
Message:
[your-message]
```

You can also add additional text or instructions to the message body if needed.

---

## 7. Messages Tab (User Feedback)

The **Messages** tab controls the messages visitors see when they submit the form.

Examples include:

* Success message
* Error messages
* Required field warnings
* Spam detection notices

You can edit these messages to match your site's tone or instructions.

Example:

```
Your message was sent successfully. Thank you!
```

---

## 8. Additional Settings Tab

This section allows you to add advanced configuration snippets.
Most basic forms do not require changes here.

---

## 9. Adding a Dropdown Field

You can add new fields to collect additional information.

Example: Add a dropdown asking how someone found your website.

### Step 1: Add a label to the form

In the **Form** tab, add:

```
How did you find our website?
```

### Step 2: Insert a dropdown field

1. Place your cursor under the label.
2. Click the **Dropdown menu** button above the editor.
3. Configure the field:

   * **Required field**: check if needed
   * **Name**: change to something descriptive (example: `referral`)
   * **Options**: add choices such as:

```
Google
Email
Friend
```

4. Click **Insert Tag**.

The generated tag will appear in the form editor.

Example:

```
[select referral "Google" "Email" "Friend"]
```

5. Click **Save**.

---

## 10. Include New Fields in Email Messages

If you add new fields, you must also add them to the **Mail tab** so the submitted data appears in the email.

Example:

```
How did you find our website?
[referral]
```

If the field is not added to the email template, the information will not appear in the message.

---

## 11. Test the Form

After making changes:

1. Visit the page containing the form.
2. Fill out the fields.
3. Submit the form.
4. Confirm the email arrives with the correct information.

---

✅ You now have a working contact form and can customize it with additional fields as needed.
